# [Arsenal Data](https://github.com/thulin82/arsenal-data)

A NodeJS/API version of [arsenal_webscrape](https://github.com/thulin82/arsenal_webscrape)

## Setup

### Get project

```
$ git clone git@github.com:thulin82/arsenal-data.git
```

### Get dependencies

```
$ npm install
```

### Run app

```
$ npm start
```
